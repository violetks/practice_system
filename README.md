# practice_system
毕设项目——Java题库管理系统

技术栈：JSP+Servlet+HTML+CSS+JS+JQuery

- 开发工具：IntelliJ IDEA 2018.3.3 x64
- 数据库：MySQL 5.6
- 辅助工具：Navicat for MySQL
- 服务器：Apache Tomcat 8.0.14
- 运行环境：Java 1.8
